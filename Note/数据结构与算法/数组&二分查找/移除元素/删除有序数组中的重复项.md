```c
int removeDuplicates(int* nums, int numsSize){
 if (numsSize == 0) {
        return 0;
    }
    int slow = 0;
    int fast = 1;
    while (fast < numsSize) {
        if (nums[fast] != nums[slow]) {
            slow++;  //先移slow再赋值
            nums[slow] = nums[fast];
        }
        fast++;
    }
    return slow + 1;
}
```

一定是先移slow再将fast赋值给它，一定要学着从错误处反推！最后是返回的slow+1也要注意。

```java
class Solution {
    public int removeDuplicates(int[] nums) {
        if (nums.length == 1) {
            return 1;
        }
        int slow = 0;
        for (int fast = 1; fast < nums.length; fast++) {
            if (nums[fast] != nums[slow]) {
                slow++;
                nums[slow] = nums[fast];
            }
        }
        return slow + 1;
    }
}
```
