```c
void moveZeroes(int* nums, int numsSize) {
    int fast = 0;
    int slow = 0;
    while(fast < numsSize){
        if(nums[fast] != 0){
            nums[slow++] = nums[fast]; //可以直接赋值
        }
        fast++;
    }
    while(slow < numsSize){
        nums[slow++] = 0;  //把后面的都赋为0
    }
}
```

要转换思想！可以先把0都删了，再在把后面多的元素全部赋为0。

```java
class Solution {
    public void moveZeroes(int[] nums) {
        if(nums.length == 1){
            return;
        }
        int slow = 0;
        for(int fast = 0; fast < nums.length; fast++){
            if(nums[fast] != 0){
                nums[slow] = nums[fast];
                slow++;
            }
        }
        while(slow < nums.length){
            nums[slow++] = 0;
        }
    }
}
```
